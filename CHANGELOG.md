# Changelog

Todas as alteracoes relevantes deste projeto serao documentadas aqui.

## 2.3.0 - 2026-05-12

- Substituido o script principal pela nova base `2.3`.
- Adicionado modo teste para simular acoes destrutivas por padrao.
- Adicionada verificacao de compatibilidade do ambiente.
- Adicionado registro de logs em `logs/`.
- Melhorada a geracao de relatorios em `relatorios/`.
- Adicionada ajuda de recuperacao de acesso com caminhos oficiais.
- Adicionada consulta de licenciamento do Windows com `slmgr`.
- Ajustada a opcao 18 para orientacao oficial de ativacao/licenciamento, sem executar ativadores, bypass, chaves ou scripts externos.
- Atualizado o README com comparativo entre a versao anterior e a versao 2.3.

## 2.0.0 - 2026-05-12

- Publicacao inicial do projeto no GitHub.
- Adicionado script principal `suporte_tecnico.py`.
- Adicionado README com requisitos, uso e observacoes de seguranca.
- Adicionada licenca MIT.
- Configurado `.gitignore` para evitar publicacao de builds, executaveis, logs, relatorios e arquivos locais.
